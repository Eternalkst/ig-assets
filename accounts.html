Заведи нужные аккаунты

Понадобятся 3 бесплатных аккаунта. Если какой-то уже есть — пропусти этот блок.

1. Facebook — нужен для Meta
A

Если аккаунта Facebook нет

Открой facebook.com → нажми Создать аккаунт.

Впиши имя, почту или телефон, придумай пароль, дату рождения — и «Регистрация».

Подтверди почту/телефон по коду. Готово.

Открыть Facebook →
B

Если аккаунт Facebook есть

Просто зайди на facebook.com и войди своим логином и паролем.

💡 Этот Facebook будет использоваться только для входа в кабинет разработчика Meta. Постить в нём ничего не нужно.
2. Instagram — сделай профессиональным
C

Переключи аккаунт в «Бизнес»

В приложении Instagram: Настройки → Тип аккаунта и инструменты → Переключиться на профессиональный → выбери Бизнес (или Автор).

Убедись, что профиль открытый (не закрытый): Настройки → Конфиденциальность → выключи «Закрытый аккаунт».

⚠️ Без профессионального и открытого аккаунта токен не сгенерируется — это обязательно.
3. GitHub — нужен для облака
D

Создай аккаунт GitHub

Открой github.com → нажми Sign up.

Впиши почту, придумай пароль и имя пользователя (латиницей, например ivan-astana).

Подтверди почту по коду. Всё — аккаунт готов, он бесплатный.

Открыть GitHub →
💡 Больше в GitHub ничего делать не надо — Claude сам создаст репозиторий и всё настроит на шаге запуска.
🌍 Продукт рассчитан на Казахстан и страны СНГ без санкционных ограничений. В некоторых странах (напр. РФ) сервисы Meta, GitHub и Claude могут быть недоступны — учитывай это.
Застрял на регистрации? Нажми в боте «🙋 Куратор» — поможем.
